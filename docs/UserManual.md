# FlowMate-Echo 用户手册

## 快速开始

### 环境要求
- Node.js >= 18.0
- Python >= 3.10
- 摄像头 (可选，用于疲劳检测)

### 安装步骤

1. **安装前端依赖**
```bash
npm install
```

2. **安装后端依赖**
```bash
pip install -r requirements.txt
```

3. **配置环境变量**
编辑 `.env` 文件，填入您的 API Keys

4. **启动应用**
```bash
# 启动后端服务
npm run start:backend

# 启动前端应用
npm run dev
```

## 功能说明

### 数字人状态
- **待机 (Idle)**: 数字人安静看书，等待您开始工作
- **工作 (Work)**: 检测到键盘活动，数字人陪您一起打字
- **专注 (Focus)**: 进入心流状态，数字人抬头关注您
- **嘘 (Shhh)**: 心流保护时刻，数字人做出嘘手势
- **休息 (Stretch)**: 提醒您该休息了

### 任务面板
- 输入任务描述，AI 自动拆解为可执行步骤
- 支持语音播报任务内容
- 完成任务后自动标记

### 心流计时器
- 自动检测工作和休息时间
- 心流豁免：深度专注时不打扰
- 强制休息：保护您的健康

## 快捷键
- `Ctrl + Shift + F`: 快速启动/暂停
- `Ctrl + Shift + B`: 切换休息模式
- `Esc`: 最小化到托盘

## 常见问题

### Q: 数字人视频不显示？
A: 请确保 `src/renderer/assets/video_loops/` 目录下有对应的视频文件

### Q: 后端连接失败？
A: 请检查后端服务是否正常启动，端口 8000 是否被占用

### Q: 语音合成不工作？
A: 请确认 `.env` 中的 API Key 配置正确
