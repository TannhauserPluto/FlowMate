# FlowMate-Echo 产品白皮书 V9.0

## 1. 产品愿景

FlowMate-Echo 是一款基于 AI 的陪伴式心流助手，通过数字人伴学、智能感知和心流保护机制，帮助用户进入并保持专注状态。

## 2. 核心功能

### 2.1 数字人伴学 (Body Doubling)
- 预渲染视频循环实现流畅的数字人陪伴
- 多种状态：待机、工作、专注、休息

### 2.2 智能感知系统
- **视觉感知**: 基于 Qwen-VL 的屏幕内容理解
- **行为感知**: 键盘活动监测判断工作状态
- **疲劳检测**: 基于 OpenCV 的面部疲劳分析

### 2.3 心流保护机制
- 心流状态机: Idle -> Work -> Flow -> Immunity -> Break
- 心流豁免: 检测到深度专注时自动延迟提醒
- 嘘手势: 标志性的心流保护动作

### 2.4 生成式任务管理
- 基于 Qwen-Max 的智能任务拆解
- 语音播报: CosyVoice 语音合成

## 3. 技术架构

### 3.1 前端技术栈
- Electron + React + TypeScript
- TailwindCSS 样式框架
- Vite 构建工具

### 3.2 后端技术栈
- Python FastAPI
- ModelScope 模型服务
- DashScope API

## 4. 目标用户

- 需要专注工作的职场人士
- 自律学习的学生群体
- 自由职业者和远程工作者

## 5. 商业模式

- 基础版: 免费使用核心功能
- 专业版: 高级 AI 功能和数据分析
- 企业版: 团队协作和管理功能
